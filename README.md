# YZClockView

A custom UIView which can display a time with two arc.
<img src="../blob/master/Demo.png" alt="YZClockView Screenshot" width="640" height="1136" />

## Requrements

- Xcode 5 or higher
- iOS 6.0 or higher

## Installation

### With CocoaPods

In your `Podfile`:

```
pod 'YZClockView'
```

## Demo

Build and run the YZClockViewDemo.xcodeproj
